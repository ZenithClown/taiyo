<p align = "justify"><b>Taiyō</b> is the romanization Japanese word representing the Sun or the Ocean - thus the name of the package <i>taiyo</i>: a package dedicated for fetching and processing data from National Solar Radiation Database (NSRDB).</p>

<p align = "justify">NSRDB <a href = "https://nsrdb.nrel.gov/data-sets/api-instructions.html">API Instructions</a> is implemented using this documentation. Keeping matter simple, this package initiates an object which can be passed to fetch data.</p>
