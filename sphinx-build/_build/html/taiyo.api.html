class taiyo.api.base_configuration¶

Bases: object

Create an Object for API Calls

For ease of access, all the attributes are to be inserted during the input() Calls

Parameters

  • user_name (str) – User Name of the User by which the API Key is Registered

  • location (list or tuple) – Location at which the Data is to be Fetched from, in the format (latitude, longitude) These are internally modified and are associated with the attribute lat and lon respectively

  • year (str) – Year of which the Data is Required to Fetch
